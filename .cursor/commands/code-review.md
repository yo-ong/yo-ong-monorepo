# Code Review Checklist

## Overview
코드 리뷰 시 확인할 항목입니다.

## 기능
- [ ] 코드가 의도대로 동작하는가
- [ ] 엣지 케이스 처리가 있는가
- [ ] 에러 처리가 적절한가
- [ ] 버그나 논리 오류가 없는가

## 코드 품질
- [ ] 가독성과 구조가 좋은가
- [ ] 함수가 작고 집중되어 있는가
- [ ] 변수명이 명확한가
- [ ] 코드 중복이 없는가
- [ ] 프로젝트 컨벤션을 따르는가

## 모노레포
- [ ] 공통 코드는 packages에 배치했는가
- [ ] @repo/ui, @repo/api-client, @repo/utils 적절히 사용했는가
